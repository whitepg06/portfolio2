Java Game of Life

Java Game of Life is a random color generating that generates colors in various different cells. The user may select the color they want the cells to appear whether they would like the cells black and white, or colorful. Other settings the user can use on this program is add further code to make the program to have a more extensive outcome such as using mathematical equations to generalize a population based off of the random color generator, giving each cell the value of a person.
